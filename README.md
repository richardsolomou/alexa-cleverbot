# alexa-cleverbot

## Environment
Add this to your `env.json`
```json
{
  "dev": {
    "cleverbot": {
      "user": "CLEVERBOT_API_USER",
      "key": "CLEVERBOT_API_KEY",
      "nick": "SESSION_NICKNAME"
    }
  },
  "release": {
    "cleverbot": {
      "user": "CLEVERBOT_API_USER",
      "key": "CLEVERBOT_API_KEY",
      "nick": "SESSION_NICKNAME"
    }
  }
}
```
